# AI-Powered-Real-Time-Market-Intelligence-and-auto-Decission-engine
AI-powered market intelligence system using Python, Machine Learning, Flask for sales analysis, prediction, and business decision support.
This project was developed as part of a Data Science & Analytics internship at ProAzure Software Solutions Pvt. Ltd.. 
The system leverages Machine Learning and Data Analytics techniques to analyze historical sales data, identify trends, and generate intelligent business insights. 
Using the Super Store dataset, the project performs data preprocessing, exploratory data analysis (EDA), predictive modeling with XGBoost, and interactive visualization through Flask-based applications.
The goal is to support data-driven decision-making by providing sales predictions, market insights, and performance analysis through an easy-to-use dashboard.

## Features
- Sales Prediction
- Profit Analysis
- Interactive Dashboard
- Region-wise Analysis
- Product Performance
- Decision Support System
